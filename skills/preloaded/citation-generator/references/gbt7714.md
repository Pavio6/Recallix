# GB/T 7714 简明参考

## 何时使用

当用户明确要求“中国国家标准引用格式”“GB/T 7714”，或需要整理中文学术文献参考文献时使用本参考。

## 常见文献类型

| 类型 | 标记 | 示例格式 |
|---|---|---|
| 专著 | `[M]` | 作者. 书名[M]. 出版地: 出版者, 年份. |
| 期刊文章 | `[J]` | 作者. 题名[J]. 刊名, 年份, 卷(期): 起止页码. |
| 学位论文 | `[D]` | 作者. 题名[D]. 城市: 学校, 年份. |
| 会议论文 | `[C]` | 作者. 题名[C]//论文集名. 出版地: 出版者, 年份: 页码. |
| 电子资源 | `[EB/OL]` | 作者. 题名[EB/OL]. (发布日期)[引用日期]. 获取和访问路径. |

## 生成规则

1. 作者后使用句点。
2. 文献题名后紧跟类型标识。
3. 出版地、出版者、年份之间使用中文冒号和逗号。
4. 期刊文章应尽量保留：
   - 年份
   - 卷号
   - 期号
   - 页码
5. 信息缺失时，不要虚构；保留已知字段，并可标注“信息未提供”。

## 示例

```text
[1] 张三. 大模型系统设计[M]. 北京: 电子工业出版社, 2025.
[2] 李四, 王五. 检索增强生成系统综述[J]. 软件学报, 2024, 35(6): 1201-1220.
[3] 赵六. 面向知识库问答的混合检索研究[D]. 上海: 复旦大学, 2023.
```

## 与默认简化格式的区别

- 默认简化格式更适合内部知识库回答中的来源标注。
- GB/T 7714 更适合正式论文、报告、参考文献清单。
- 如果用户只要求“注明来源”，不要擅自升级为 GB/T 7714。
